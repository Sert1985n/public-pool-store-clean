# Icon status

Локальные `icon.png` добавлены в папки `Apps/<app>/`.

Что сделано:
- для рабочих монет использованы иконки из архива `img.zip`
- для `emark`, `pirate`, `hush`, `komodo`, `zcash`, `peercoin` тоже поставлены локальные иконки
- для монет без подходящей иконки в архиве оставлен `generic.png`

После загрузки в GitHub поля `icon:` будут работать через `raw.githubusercontent.com`.
